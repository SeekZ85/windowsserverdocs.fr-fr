# <a name="use-the-my-server-app-to-connect-to-windows-server-essentials"></a>Utilisez l’application My Server pour se connecter à Windows Server Essentials

>S’applique à: Windows Server2016Essentials, Windows Server2012R2 Essentials, Windows Server2012Essentials

Les applications mon serveur pour Windows Server Essentials vous permettent de se connecter aux ressources et d’effectuer de petites tâches d’administration sur votre serveur Windows Server Essentials à partir de votre ordinateur portable, un PC ou un appareil Surface. Dans My Server, vous pouvez gérer les utilisateurs, appareils et des alertes et utiliser des fichiers partagés sur le serveur. Lorsque vous êtes hors connexion, vous pouvez continuer à travailler avec les fichiers récemment consultés dans My Server; vos modifications hors connexion sont automatiquement synchronisées avec le serveur la prochaine fois que vous vous connectez.  
  
 Si votre serveur exécute le système d’exploitation Windows Server Essentials, utilisez l’application My Server d’origine. Si votre serveur exécute le système d’exploitation Windows Server Essentials, vous devez utiliser l’application My Server 2012 R2 mise à jour. Les deux applications peuvent être téléchargées à partir de [d’applications pour Windows](https://windows.microsoft.com/windows-8/apps).  
  
> [!TIP]

>  Pour accéder aux ressources dans Windows Server Essentials à partir de votre Windows Phone, utilisez l’application My Server (pour Windows Server Essentials) ou l’application téléphonique My Server 2012 R2 (pour Windows Server Essentials). Pour plus d’informations sur l’application téléphonique My Server, consultez [utiliser l’application My Server pour Windows Phone](Work-Remotely-in-Windows-Server-Essentials.md#BKMK_2). Pour plus d’informations sur l’application téléphonique My Server 2012 R2, voir le billet de blog [applications My Server 2012 R2 Windows et Windows Phone](http://blogs.technet.com/b/sbs/archive/2013/11/19/my-server-2012-r2-windows-and-windows-phone-apps.aspx).

##  <a name="BKMK_WhatsNew"></a>Quelles s nouveau dans My Server 2012 R2  
 Outre les fonctionnalités disponibles dans My serveur, My Server 2012 R2 offre les nouvelles fonctionnalités suivantes pour les clients avec Windows Server Essentials:  
  
-   **Accès à des sites d’équipe SharePoint Online et bibliothèques** -l’intégration de tirer parti de Microsoft Office 365 avec Windows Server Essentials pour accéder aux sites d’équipe SharePoint Online et ouvrir des documents dans vos bibliothèques SharePoint Online dans My Server 2012 R2.  
  
-   **Les connexions Bureau à distance pour les ordinateurs clients et serveur** -utilisation **se connecter à distance** dans My Server 2012 R2 pour se connecter à votre serveur Windows Server Essentials et les ordinateurs clients à l’aide des services Bureau à distance.  
  
##  <a name="BKMK_OS"></a>Configuration requise du système d’exploitation  
 Pour utiliser My Server ou My Server 2012 R2 sur votre ordinateur portable, un PC ou un appareil Surface, l’ordinateur doit avoir un des systèmes d’exploitation suivants: Windows 8.1, Windows RT 8.1, Windows 8 ou Windows RT.  
  
##  <a name="BKMK_Install"></a>Installer My Server  
 Vous pouvez télécharger l’application My Server appropriée à partir de la [applications pour Windows](https://windows.microsoft.com/windows-8/apps) stocker. Si votre ordinateur possède le système d’exploitation Windows 8 ou Windows RT, et que vous souhaitez vous connecter au serveur sur votre intranet en utilisant le nom de serveur, vous devez également installer un certificat à partir du serveur.  
  
#### <a name="to-install-my-server-or-my-server-2012-r2-on-your-windows-based-laptop-pc-or-surface-device"></a>Pour installer My Server ou My Server 2012 R2 sur votre portable Windows, un PC ou un appareil Surface  
  
1.  Installez l’application My Server appropriée à partir de [d’applications pour Windows](https://windows.microsoft.com/windows-8/apps) sur votre portable Windows, un PC ou un appareil Surface.  
  
    |Système d’exploitation serveur|Téléchargement|  
    |-----------------------------|--------------|  
    | Windows Server Essentials|[Mon serveur](https://apps.microsoft.com/webpdp/app/19d94f81-db21-4234-8b49-806694dbbaea)|  
    | Windows Server Essentials|[Mon serveur 2012 R2](https://apps.microsoft.com/webpdp/app/67e86695-bda3-4f32-96c4-2e20e56f1cf3)|  
  
2.  Si vous voulez être en mesure d’utiliser le nom du serveur pour se connecter à votre serveur Windows Server Essentials sur un intranet, et que votre appareil dispose du système d’exploitation Windows 8 ou Windows RT, vous pouvez installer le certificat CAROOT.cer par défaut sur votre ordinateur en effectuant la procédure suivante. Installation manuelle d’un certificat n’est pas nécessaire dans Windows 8.1 et Windows RT 8.1.  
  
###  <a name="BKMK_InstallCert"></a>Pour installer un certificat pour My Server sur votre appareil Windows 8, Windows 8.1 ou Windows RT  
  
1.  Ouvrez un navigateur web sur votre ordinateur et télécharger le certificat par défaut, CAROOT.cer, à partir du serveur. Pour ce faire, tapez la commande suivante, en remplaçant le nom de votre serveur (par exemple, **marketing.contoso.com**) pour <*nom_serveur*>:  
  
     **http://<servername\>/Connect/default.aspx?Get=caroot.cer**  
  
2.  Installer le certificat:  
  
    1.  Sur le **Démarrer** page, ouvrez **Explorateur de fichiers**.  
  
    2.  Assurez-vous que les éléments masqués et les extensions de nom de fichier sont affichées: sur le **affichage** onglet le **afficher/masquer** groupe, sélectionnez le **éléments masqués** et **extensions de nom de fichier** cases à cocher.  
  
    3.  Accédez au dossier qui contient le fichier CAROOT.cer que vous venez de télécharger.  
  
    4.  Cliquez sur le fichier CAROOT.cer, puis cliquez sur **ouvrir**.  
  
    5.  Dans le **certificat** boîte de dialogue, cliquez sur **installer le certificat**.  
  
    6.  Sélectionnez **Local Machine** comme emplacement d’installation, puis cliquez sur **suivant**.  
  
    7.  Sur le **magasin de certificats** page de l’Assistant, sélectionnez **placer tous les certificats dans le magasin suivant**et utiliser **Parcourir** pour choisir le **Trusted Root Certification Authorities** stocker. Puis cliquez sur **Terminer**.  
  
##  <a name="BKMK_UseServer"></a>Utiliser mon serveur  
 Pour commencer à l’aide de l’application My Server ou My Server 2012 R2, ouvrez l’application et effectuez une visite rapide de ses fonctionnalités.  
  
#### <a name="to-open-my-server-or-my-server-2012-r2"></a>Pour ouvrir My Server ou My Server 2012 R2  
  
1.  Ouvrez My Server (pour Windows Server Essentials) ou My Server 2012 R2 (pour Windows Server Essentials) à partir de la **Démarrer** page de votre appareil Windows.  
  
2.  Connectez-vous au serveur Windows Server Essentials à l’aide de votre compte sur le serveur. Pour identifier le serveur:  
  
    -   Dans un environnement hors site, utilisez par exemple, le nom de domaine du serveur s **marketing.contoso.com**.  
  
    -   Si vous vous connectez via un intranet local, utilisez le nom du serveur dans l’exemple précédent, le nom du serveur est **marketing**. (Si vous utilisez un ordinateur Windows 8 ou Windows RT pour vous connecter sur site, et que vous souhaitez utiliser cette méthode, vous devez installer un certificat CAROOT.cer à partir du serveur. Pour plus d’informations, voir [pour installer un certificat pour My Server sur votre appareil Windows 8, Windows 8.1 ou Windows RT](#BKMK_InstallCert).)

##  <a name="BKMK_ConnectServer"></a>Comment se connecter à votre serveur à partir de votre réseau local  
 Pour utiliser correctement l’application mon serveur dans Windows Server Essentials pour Windows Phone, Windows 8 et Windows 8.1, vous devez tout d’abord installer le certificat de serveur sur votre appareil. Cela connecte l’appareil à votre serveur exécutant Windows Server Essentials dans votre réseau local. Utilisez les procédures suivantes pour vous connecter à votre serveur dans votre réseau local et installer le certificat de serveur sur votre Windows Phone ou un ordinateur qui exécute Windows 8 ou Windows 8.1.  
  
#### <a name="to-connect-to-your-server-from-your-windows-phone"></a>Pour se connecter à votre serveur à partir de votre Windows Phone  
  
1.  Sur votre Windows Phone qui exécute Windows 8 ou Windows 8.1, ouvrez Internet Explorer.  
  
2.  Dans la barre d’adresses, tapez **http://<yourservername\>/connect/default.aspx?get=caroot.cer**, puis appuyez sur ENTRÉE.  
  
3.  Lorsque vous êtes invité à installer le certificat caroot.cer, cliquez sur **installer**.  
  
4.  Attendez que l’installation du certificat terminer.  
  
    > [!NOTE]
    >  Une fois l’installation du certificat terminée, vous pouvez vous connecter l’application mon serveur pour Windows Phone en utilisant le nom de votre serveur et vos informations d’identification réseau.  
  
#### <a name="to-connect-to-your-server-in-your-local-network-from-a-computer-running-windows-8-or-windows-81"></a>Pour se connecter à votre serveur dans votre réseau local à partir d’un ordinateur exécutant Windows 8 ou Windows 8.1  
  
1.  Sur un ordinateur qui exécute Windows 8 ou Windows 8.1, ouvrez Internet Explorer.  
  
2.  Dans la barre d’adresses, tapez **http://<yourservername\>/connect/default.aspx?get=caroot.cer**, puis appuyez sur ENTRÉE.  
  
3.  Lorsque vous êtes invité à installer le certificat caroot.cer, cliquez sur **ouvrir**.  
  
4.  Dans le **certificat** boîte de dialogue, cliquez sur **installer le certificat**.  
  
5.  Dans l’Assistant Importation de certificat, sélectionnez **Local Machine** comme emplacement du magasin.  
  
6.  Sélectionnez **placer tous les certificats dans le magasin suivant**, puis accédez à la **Trusted Root Certification Authorities** emplacement.  
  
7.  Suivez les instructions pour terminer l’Assistant.  
  
    > [!NOTE]
    >  Une fois l’installation du certificat terminée, vous pouvez vous connecter l’application mon serveur pour Windows 8 ou Windows 8.1 à l’aide de votre nom de serveur et vos informations d’identification réseau.
