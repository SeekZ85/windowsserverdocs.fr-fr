# <a name="bitsadmin-help"></a>bitsadmin help



Affiche l’utilisation de ligne de commande.

## <a name="syntax"></a>Syntaxe

```
bitsadmin /help | /?
```

## <a name="BKMK_examples"></a>Exemples

L’exemple suivant récupère l’aide en ligne de commande.
```
C:\>bitsadmin /help
```
