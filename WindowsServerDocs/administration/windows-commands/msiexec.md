# <a name="msiexec"></a>msiexec



## <a name="msiexec"></a>msiexec

Fournit les moyens de les installer, modifier et effectuer des opérations sur le programme d’installation de Windows à partir de la ligne de commande.

Pour la syntaxe et des exemples montrant comment utiliser cette commande, consultez [Msiexec](https://go.microsoft.com/fwlink/?LinkId=94329).
