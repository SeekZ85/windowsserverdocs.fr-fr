# <a name="troubleshooting-activation-error-codes"></a>Dépannage des Codes d’erreur d’Activation

**Utilisateurs à domicile** cet article est destiné à l’utilisation par les agents de support et de professionnels de l’informatique. Si vous recherchez plus d’informations sur les messages d’erreur d’activation Windows, consultez les sites Web Microsoft suivants :

[Obtenir de l’aide avec des erreurs d’activation de Windows](https://support.microsoft.com/help/10738/windows-10-get-help-with-activation-errors) 

Lorsque vous essayez d’utiliser une clé d’Activation Multiple (MAK) ou le Service de gestion de clés (KMS) pour effectuer l’Activation en Volume sur un ou plusieurs ordinateurs en fonction de Windows, vous recevez des messages d’erreur qui contiennent des codes d’erreur spécifiques. Cet article explique comment résoudre ces erreurs.
