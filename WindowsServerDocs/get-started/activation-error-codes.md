# Résolution des problèmes de Codes d’erreur d’Activation

**Utilisateurs à domicile** Cet article est destiné à utiliser par les agents du support et les professionnels de l’informatique. Si vous recherchez pour plus d’informations sur les messages d’erreur de l’activation de Windows, accédez aux sites Web Microsoft suivants:

[Obtenir de l’aide sur les erreurs d’activation de Windows](https://support.microsoft.com/help/10738/windows-10-get-help-with-activation-errors) 

Lorsque vous essayez d’utiliser une clé d’Activation Multiple (MAK) ou le Service de gestion des clés (KMS) pour effectuer l’Activation en Volume sur un ou plusieurs ordinateurs basé sur Windows, vous recevez des messages d’erreur qui contiennent des codes d’erreur spécifiques. Cet article explique comment résoudre les problèmes liés à ces erreurs.
