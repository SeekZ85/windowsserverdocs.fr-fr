# <a name="troubleshooting-activation-error-codes"></a>Résolution des codes d’erreur d’activation

**Utilisateurs domestiques** Cet article est destiné aux professionnels de l’informatique et agents d’assistance. Si vous recherchez plus d’informations sur les messages d’erreur d’activation Windows, consultez les sites web Microsoft suivants :

[Obtenir de l’aide sur les erreurs d’activation de Windows](https://support.microsoft.com/help/10738/windows-10-get-help-with-activation-errors) 

Lorsque vous essayez d’utiliser une clé d’activation multiple (MAK) ou le service de gestion de clés (KMS) pour effectuer une activation en volume sur un ou plusieurs ordinateurs Windows, vous recevez des messages d’erreur qui contiennent des codes d’erreur spécifiques. Cet article explique comment résoudre ces erreurs.
