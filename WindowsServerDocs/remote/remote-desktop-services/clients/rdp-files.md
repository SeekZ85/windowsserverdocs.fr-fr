# <a name="supported-remote-desktop-rdp-file-settings"></a>Prise en charge des paramètres de fichier RDP Bureau à distance

Le tableau suivant contient la liste des paramètres de fichier RDP pris en charge que vous pouvez utiliser avec les clients Windows et HTML. Un « x » dans la colonne de la plateforme indique que le paramètre est pris en charge. Toutefois, cette liste n’est pas une liste complète des paramètres pris en charge pour les clients Windows et HTML5. Nous continuerons à mettre à jour cette table pour inclure plus des paramètres RDP pris en charge pour les clients Windows et HTML5, ainsi que macOS, iOS et Android clients.

| autre adresse de complète : s:value | Spécifie un autre nom ou l’adresse IP de l’ordinateur distant. | N’importe quel nom valide ou une adresse IP de l’ordinateur distant, tel que « 10.10.15.15 » | | x | x | x |
| autre shell : s:value        | Détermine si un programme démarre automatiquement lorsque vous vous connectez à RDP. Ce paramètre correspond au chemin d’accès et le fichier boîte Nom du programme sur l’onglet programmes du client Connexion Bureau à distance. Pour spécifier un autre interpréteur de commandes, entrez un chemin valide vers un fichier exécutable pour la valeur, tels que « C:\ProgramFiles\Office\word.exe ». Ce paramètre détermine également quel chemin d’accès ou l’alias de l’Application à démarrer au moment de la connexion si RemoteApplicationMode est activé à distance. | "C:\ProgramFiles\Office\word.exe" || x | x | x |
